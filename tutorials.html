Copy and Paste in VIM on Windows

To copy and paste into VIM in Windows follow these steps:

  1. In VIM, set the paste option using set paste
  2. Copy the text by selecting it and pressing ctrl+c
  3. In VIM, move the cursor to where you want to paste the text and right click.
Done.
