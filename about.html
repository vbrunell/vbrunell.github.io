Software engineer living in Orlando, Florida.


MS in Computer Science from Florida State University. My thesis research involved code optimization using an optimizing compiler and a new computer architecture framework that took advantage of identifying and collapsing data dependencies among sets of three-address machine code instructions to increase instruction-level parallelism. The areas of computer science that interest me include data engineering, data science, compilers, computer architecture, programming language design, and machine learning.

Before entering the CS program at FSU, I spent some time in Japan working for the Ministry of Education as a high school English teacher and coordinator for the English teaching program in Tokushima Prefecture.
